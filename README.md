 lagi pengen build AI automated trader sendiri, ini API stack yang mungkin di butuhin.

> TradingView Lightweight Charts http://github.com/tradingview/lightweight-charts Library charting resmi dari TradingView. 14K+ stars, cuma 45KB, gratis. Buat visualisasi harga real-time.

> CCXT http://github.com/ccxt/ccxt Satu library, konek ke 100+ exchange. Binance, Bybit, OKX, Hyperliquid, semua lewat satu API. Support Python, JS, TS, Go, PHP.

> Binance API http://binance-docs.github.io/apidocs API resmi Binance. Eksekusi order, market data, WebSocket streaming. Dokumentasi paling lengkap dan komunitas paling gede.

> Hyperliquid Python SDK http://github.com/hyperliquid-dex/hyperliquid-python-sdk SDK resmi buat trading perps on-chain. Buat kalian yang mau automated trading di DEX.

> Tavily API http://tavily.com Search engine khusus AI. Buat scraping berita dan sentimen market real-time. Hasilnya udah terstruktur, tinggal diproses AI agent kalian.

> RTK http://github.com/rtk-ai/rtk CLI proxy yang potong token AI 60-90%. Rust, single binary. Hemat cost kalau pake Claude/GPT buat proses data market.
